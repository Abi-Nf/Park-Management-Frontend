# Park Management UI
This repository contains the user interface of our project: `Park Management`

The deployment link is [here](https://park-manager.vercel.app/)

## Running the app
- with *npm*
```sh
# development
$ npm start
```
- with *yarn*
```sh
# development
$ yarn start
```
## Test
- with *npm*
```sh
$ npm test
```
- with *yarn*
```sh
$ yarn test
```
